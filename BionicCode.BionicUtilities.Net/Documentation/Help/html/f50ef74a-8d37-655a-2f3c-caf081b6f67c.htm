FilePathFilterFilterFilePathsFromMixedPathsIgnoringFolders Method

Extracts valid paths or paths with a specified extension from a collection of paths. The path collection can be a mix-up of files and folders. In case the path describes a folder, the filter will ignore it including all containing files.
